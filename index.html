Fully Featured Backup Suite

Designed natively in Kotlin to provide professional-grade camera media synchronization and direct file orchestration.

🔌

USB / MTP Hardware Sync

Plug your camera (Canon, Nikon, Sony, GoPro) directly into your Android device via OTG. Stream media seamlessly using Android's native MTP stack.

☁️

Concurrent Multi-Cloud

Transmit files in parallel directly to Google Drive, Nextcloud, and Immich simultaneously, configured and routed cleanly by a unified dispatch ledger.

📂

Local Storage Fallback

No physical camera on hand? Seamlessly scan and upload your pre-configured local storage folder via the secure Android Storage Access Framework (SAF).

⚖️

Dynamic Routing Matrix

Room-backed matrix lets you route common media (JPEGs/PNGs) and RAW/support files (.xmp, .lrv, .thm) to separate custom cloud paths.

📶

Cellular Data Gates

Have strict billing constraints? Tweak per-provider metered cellular gates to independently authorize background transfers exclusively on Wi-Fi.

🛡️

Zero-Footprint Mode

Upload files securely to cloud storage without leaving heavy local cache records, automatically handling memory deletion post-sync.

About The Developer

Hello, I'm **purnadika**! I built Potopopi CamSync out of a passion for photography and private, self-hosted technology. Dissatisfied with the lack of reliable direct camera-to-cloud Android clients, I developed a native, privacy-first companion client that runs perfectly on modern mobile phones.

The project is free and built entirely around complete user privacy. We will never sell, store, or intercept your high-res files or account credentials.
